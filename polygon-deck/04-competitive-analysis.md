## The Pattern: "API" Is the Trigger

Across all 200+ probes, one word consistently triggers ads — regardless of what the rest of the conversation is about.

| Prompt | Company | What They Sell | Market Cap |
|---|---|---|---|
| "crypto trading platform with an **API** that lets me automate orders" | Cursor | AI code editor | Widely adopted |
| "crypto exchange **API** gives me the best access to on-chain data" | MongoDB | Enterprise database | $20B+ |
| "commodities trading **API** that supports futures contracts" | Anakin (Wire) | Web-to-API tool | Venture-backed |
| "exchange **API** with a sandbox environment and paper trading" | Tinfoil | Encrypted AI inference | Startup |

Cursor appeared on a crypto trading prompt. It's an AI code editor — not a trading platform. MongoDB appeared on a DeFi prompt. It's a database — not a crypto company. Neither is bidding on their own category. They're bidding on "API" because they know their developer audience uses that word in conversation, and that's where the buying intent lives.

For Polygon OMS — a stablecoin payments API — this is the exact signal to own. The fintech PMs and marketplace CTOs researching "regulated stablecoin settlement API" and "fiat on-ramp API for payments" are using the same language that already triggers ads from Mastercard, MongoDB, Robinhood, and Cursor. The signal is proven. Polygon should be in that conversation before the auction gets crowded.
